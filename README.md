# CharacterImageAndJigsaw

字符画，灰度图，小图拼大图

很糊，很junk，简单的抠图加上对比色块替换。

程序使用JavaFx， IntelliJ IDEA编写，在jdk1.8以上编译

字符画使用字符 

    String[] arr={"M","N","H","Q","$","O","C","?","7",">","!",":","–",";","."};
